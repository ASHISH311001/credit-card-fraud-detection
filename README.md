# Credit Card Fraud Detection

Anomaly Detection for Credit Card Fraud using Isolation Forest and Autoencoders. Handles highly imbalanced datasets with high recall and precision.

## Overview
Production-ready anomaly detection system for identifying fraudulent credit card transactions using unsupervised machine learning techniques.

## Features
- Isolation Forest for anomaly detection
- Autoencoder neural network implementation
- Handles highly imbalanced datasets
- High recall and precision metrics
- Real-time fraud detection capabilities

## Dataset
Using Credit Card Fraud Detection dataset from Kaggle
